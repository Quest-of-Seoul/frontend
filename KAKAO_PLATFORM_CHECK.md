# 카카오맵 플랫폼 설정 확인 방법

## 현재 상황
- WebView로 카카오맵 구현
- JavaScript API 키 사용 중: `916a7bbcb8737b9ec08d7581962c6237`
- "Kakao SDK not loaded" 에러 발생

## 확인해야 할 것

### 1. 카카오 개발자 콘솔 플랫폼 설정

https://developers.kakao.com 접속 후:

1. **내 애플리케이션** → 해당 앱 선택
2. **앱 설정** → **플랫폼** 메뉴
3. **웹 플랫폼 추가** 확인:
   - 사이트 도메인: `*` (모든 도메인 허용) 또는
   - 사이트 도메인: `localhost`, `file://` 추가

### 2. JavaScript 키 확인

- **앱 설정** → **앱 키** 메뉴에서
- **JavaScript 키** 항목 확인
- 현재 키가 JavaScript 키인지 확인: `916a7bbcb8737b9ec08d7581962c6237`

## WebView가 폰에서 작동하는 이유

- WebView는 앱 안에 내장된 **미니 브라우저**입니다
- 웹 페이지처럼 작동하므로 **JavaScript 키**만 있으면 됩니다
- 네이티브 앱 키는 필요 없습니다!

## 다음 단계

1. ✅ JavaScript 키 확인
2. ✅ 웹 플랫폼 도메인 설정 (`*` 또는 `file://` 추가)
3. 📱 앱 새로고침 (Expo Go에서 Reload)
4. 🔍 콘솔 로그 확인

## 만약 계속 안 되면?

대안으로 **네이티브 SDK 대신 REST API 방식**도 가능하지만,
WebView가 가장 간단하고 빠른 방법입니다.
